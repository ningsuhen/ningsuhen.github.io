First Blog Post With Octopress

Apr 26th, 2014

Nothing much to expect guys. This is my first blog post using octopress. If you don’t know about octopress, it’s a blogging framework for hackers. Today, I’m going to explore the plugins of Octopress.

To test the plugins, I’ll use them in this post directly. Here we go.

Excerpts

The plugin allows you to display only a small portion of the post and add a “read more” link. For that, i need to use a <!—more—> tag in the content. Let’s try it out.

If you saw a link that says “Read On”, the excerpts plugin is working.

HTML5 Video Tag

Next is HTML5 Video Tag. To try this, I’m gonna use the video of sintel. One of my favorite independent films. Unfortunately, the video is not working for me. It looks like an outdated plugin. Btw, you can watch the movie @ www.sintel.org.

Image Tag

Now, I’m going to try the image tag.

Block quote

Another interesting plugin is the Block Quote. You can add a simple quote or tweets, articles,etc. Here we go.

Another Interesting reason why our Github Profile matters.

Pull quote

Pull quote allows you to display text without including them in the html markup. One special use case of a pull quote is when you want to repeat the points of a long article. e.g. We have seen examples of Excerpts, HTML5 Video Tags, Image Tag, Block Quote, Pull Quote.

Code Blocks

Here comes the most interesting part. Octopress allows you to insert code blocks without much hassle. Apparently, the additional options – start, mark & lineos are not available in master branch yet.

Gist Tag

Another feature you will love is the Gist Tag. You can insert a gist by specifying the gist id.

Include Code

This one allows you to embed code from a file in your system.

Render Partial

Render Partial allows embedding of any file into the blog post.

That wasn’t so bad. Sharing code is so easy now. Well, if you are a hacker or a programmer, i recommend you to try it right away. You can start your own blog with 0 investment. And it does make your life easier.
